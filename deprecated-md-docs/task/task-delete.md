## Удалить задачку по ID
> `DELETE /api/tasks/id` - этот запрос удалит 1 задачку, если она существует:
###### Успешный ответ (204 No Content)
<br>

Если задачка не найдена по `id`, то сервер снова ответит 204-ым кодом:
###### Успешный ответ (204 No Content)
